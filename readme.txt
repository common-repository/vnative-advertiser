vNative enables to track and setup native marketing campaigns in real-time on CPC, CPA, CPM model.

For more information visit: http://vnative.com

== Description ==

A plugin to manage ALL your cost per accusition, cost per lead, cost per sale campaigns tracking pixel and conversion pixels. Compatible with Facebook Ads, Google Adwords, WooCommerce, Easy Digital Downloads, WP eCommerce.

You can also run various CPA, CPC, CPS etc campaigns from various networks into native on your website through this plugin

== Installation ==

1. Upload `vNative` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Insert the publishers id from vnative.com dashboard to plugin settings

== Frequently asked questions ==

= Is the service paid? =

No, It's free up to 10,000 actions per month, which maybe clicks, impressions of native advertising etc

= Where do I get the publisher id? =

You need to register on vnative.com or contact your network admin,
if not then support@vnative.com
